= Assistant graphique de configuration Mesa pncconf[[cha:Assistant-graphique-Mesa]](((Assistant graphique de configuration Mesa pncconf)))(((pncconf Wizard)))

Le Point n Cliquez Configurateur couvre Mesa cartes et port parallèle
configurations stepper. Si vous avez un simple port parallèle stepper
la machine de l'Assistant Stepconf pourrait être un meilleur choix. 
Le Point n Cliquez sur Configurateur peut générer certaines 
configurations avancées sans savoir rien sur HAL.

Démarrez le programme depuis le menu PnCConf CNC ou d'une fenêtre de terminal
avec `pncconf. Pour plus d'informations sur la fenêtre de terminal, voyez le
Linux section FAQ.
