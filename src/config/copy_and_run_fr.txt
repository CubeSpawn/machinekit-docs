= Démarrage d'EMC

== Lancer emc

Après installation, emc2 se lance comme un autre programme Linux:
depuis un terminal en passant la commande `emc`, 
ou depuis le menu Applications -> CNC.

== Sélecteur de configuration

Par défaut, la boîte de dialogue de configuration Sélecteur
s'affiche lorsque vous exécutez d'abord EMC2.
Vos propres configurations personnalisées sont affichées en haut de la liste,
suivie par des exemples de configurations.
Parce que chaque configuration de l'échantillon est pour un autre type d'interface matérielle,
presque tous ne fonctionnera pas sans le matériel installé.
Les configurations répertoriées sous la catégorie "sim" entièrement géré
sans matériel ci-joint. 

Figure <<cap:Selecteur-de-configuration>> montre l'apparence 
de la fenêtre du sélecteur de configuration.

.Sélecteur de configuration pour EMC2[[cap:Selecteur-de-configuration]]

image::images/configuration-selector1_fr.png[]

Cliquez sur l'une des configurations répertoriées
pour afficher des informations spécifiques à ce sujet.
Double-cliquez sur une configuration ou cliquez sur OK
pour démarrer la configuration.
Sélectionnez "Créer un raccourci bureau" puis cliquez sur OK
pour ajouter une icône sur le bureau Ubuntu
pour lancer directement cette configuration
sans montrer l'écran de configuration Selector.

Si vous sélectionnez une configuration de l'échantillon Configurations la section,
il vous sera demandé de copier la configuration
à un emplacement dans votre répertoire home.
Il est sûr de dire «oui» à cette invite.
Si vous dites «non», EMC2 peut démarrer mais se comporter bizarrement - par exemple,
décalages entré avec Off tactile sera effacé de façon inattendue.
C'est parce que les fichiers sont en lecture seule
si vous n'avez pas d'enregistrer une copie de votre répertoire home.

== Les prochaines étapes dans la configuration

Après avoir trouvé l'exemple de configuration qui utilise
le matériel même interface que votre machine,
et l'enregistrement d'une copie de votre répertoire personnel,
vous pouvez la personnaliser en fonction des détails de votre machine.
Consultez le Manuel de l'intégrateur pour les sujets de la configuration.
