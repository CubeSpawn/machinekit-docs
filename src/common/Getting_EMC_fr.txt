= A propos d'EMC2

== Le Logiciel

* EMC2 (commande de la machine améliorée) est un système logiciel pour ordinateur
  le contrôle de machines-outils telles que fraiseuses et tours.
* EMC2 est un logiciel libre avec code source ouvert. Les versions actuelles d'EMC
  sont entièrement sous licence GNU Lesser General Public et de
  GNU General Public License (GPL et LGPL)
* EMC2 propose:
** Une interface utilisateur graphique (en fait plusieurs interfaces au choix)
** Un interprète pour "G-code" (RS-274 langage de programmation de machines-outils)
** Un système temps réel de planification de mouvement avec des look-ahead
** Le fonctionnement de l'électronique machine de bas niveau tels que les capteurs et les moteurs
** Un outil facile à utiliser "breadboard" calque pour créer rapidement un unique
   la configuration de votre machine
** Un automate programmable par logiciel avec des schémas à contacts
** Installation facile avec un Live-CD
* Il ne fournit pas de dessin (CAO - Conception Assistée par Ordinateur) ou G-code
  la génération du dessin (CAM - Computer Automated Manufacturing)
  fonctions.
* Il peut déplacer simultanément jusqu'à 9 axes et supporte une variété de
  interfaces.
* Le contrôle peut fonctionner servos vrai (analogique ou PWM) avec les commentaires
  en boucle fermée par le logiciel EMC à l'ordinateur, ou en boucle ouverte avec
  «step-servos» ou moteurs pas à pas.
* Fonctions de contrôle de mouvement incluent: rayon de la fraise et la longueur
  écart de trajectoire d'indemnisation, limitée à une tolérance spécifiée, tour
  filetage, mouvement de l'axe synchronisé, adaptative avance, nourrir l'opérateur
  remplacer, et le contrôle de vitesse constante.
* Support des systèmes de mouvement non cartésienne est fourni via personnalisée
  modules cinématique. Architectures disponibles incluent hexapodes (Stewart
  plates-formes et des concepts similaires) et les systèmes avec des joints rotatifs pour
  fournir un mouvement tels que des robots PUMA ou SCARA.
* EMC2 fonctionne sous Linux en utilisant des extensions temps réel.

== Le système d'exploitation

Ubuntu a été choisie car elle s'intègre parfaitement dans l'Open Source
vues d'EMC2:

  - Ubuntu sera toujours gratuit, et il n'ya aucun frais supplémentaire pour
    le "Enterprise Edition", nous faisons nos meilleurs travaux disponibles pour
    tout le monde sur les mêmes termes gratuit.
  - EMC2 est jumelé avec les versions LTS d'Ubuntu qui apportent un soutien
    correctifs de sécurité et de l'équipe Ubuntu pour 3 - 5 ans.
  - Ubuntu utilise le meilleur de traductions et de l'accessibilité
    l'infrastructure que la communauté du logiciel libre a à offrir, à faire
    Ubuntu utilisable pour autant de personnes que possible.
  - La communauté Ubuntu est entièrement souscrit aux principes du libre
    développement de logiciels, nous encourageons les gens à utiliser des logiciels open source,
    l'améliorer et le transmettre.

== IRC

IRC signifie Internet Relay Chat.
Il s'agit d'une connexion en direct avec d'autres utilisateurs EMC.
Le EMC canal IRC # est EMC sur freenode.

La manière la plus simple d'aller sur IRC est d'utiliser 
le client Java embarqué sur cette page:

http://www.linuxcnc.org/content/view/4/8/lang,en/[http://www.linuxcnc.org/content/view/4/8/lang,en/].

Un peu d'éthique sur le canal IRC:

 - Posez des questions précises... Évitez "quelqu'un peut m'aider?", 
   "Il ne fonctionnera pas" type de questions. 
 - Si vous êtes vraiment nouveau à tout cela, imaginez un peu votre question 
   avant qu'il ne frappe. Assurez-vous donner suffisamment d'informations pour 
   que quelqu'un puisse résoudre vos questions. 
 - Avoir de la patience quand on attend une réponse, il faut parfois une 
   alors à formuler une réponse ou tout le monde pourrait être occupé à travailler ou à 
   quelque chose. 
 - Configurez votre compte IRC avec votre nom unique afin que les gens savent qui 
   vous êtes. Si vous utilisez le client java, utiliser le même nom chaque fois que vous 
   vous connecter Cela aide les gens se rappeler qui vous êtes et si vous avez été sur 
   avant que beaucoup se souviendront des discussions passées qui fait gagner 
   du temps sur les deux extrémités.

== Liste de diffusion

Une liste de diffusion sur Internet est un moyen de poser des questions pour tout le monde sur le
cette liste pour voir et répondre à leur convenance. Vous obtiendrez de meilleurs
exposition à vos questions sur une liste de diffusion que sur l'IRC, mais
prendre plus de réponses. En un mot, vous e-mail un message à la liste et
soit obtenir quotidienne digère ou les réponses individuelles retour selon le
vous avez configuré votre compte.

Information à la liste de diffusion des utilisateurs d'EMC2 à: 

link:[https://lists.sourceforge.net/lists/listinfo/emc-users,emc-users liste de diffusion].

=== Wget Windows (((Wget Windows)))

Le programme wget est également disponible pour Windows depuis: 

http://gnuwin32.sourceforge.net/packages/wget.htm

Suivez les instructions de la page web pour télécharger et installer
le programme wget sous Windows.

Pour lancer wget ouvrez la fenêtre d'invite de commande.

Dans la plupart des Windows elle est dans Programmes/Accessoires/Commande

Naviguez jusqu'au répertoire dans lequel s'est installé wget.

Habituellement il est dans C:\Program Files\GnuWin32\bin si 
c'est le cas, tapez la commande:

`cd C:\Program Files\GnuWin32\bin`

et le prompt devrait changer pour: `C:\Program Files\GnuWin32>`

Tapez les commandes wget dans la fenêtre et pressez Entrée comme précédemment.

== Graver le CD

EMC2 est distribué sous la forme d'un fichier image de CD, appelée image ISO. 
Pour installer EMC2, vous devez d'abord graver cette image ISO sur un CD. 
Vous devez disposer d'un graveur de CD/DVD et d'un CD vierge de 80 minutes (700 Mio). 
Si la gravure du CD échoue, essayez de le graver à une vitesse plus lente.

=== Sous Linux

Avant de graver un CD, il est fortement recommandé de vérifier la
somme de contrôle md5 (hash) du fichier de l'image iso.

Ouvrez un terminal. 
Dans Ubuntu il est dans Applications/Accessoires/Terminal.

Allez dans le répertoire contenant l'image ISO précédemment téléchargée.

`cd répertoire_de_l'image`

Puis lancez la commande md5sum suivie du nom du fichier, exemple:

`md5sum -b ubuntu-10.04-linuxcnc1-i386.iso`

La commande md5sum doit retourner une simple ligne après 
le calcul de la somme de contrôle. 
Sur une machine lente le calcul peut prendre plusieurs minutes: 

`5283b33b7e23e79da1ee561ad476b05f *ubuntu-10.04-linuxcnc1-i386.iso`

Maintenant le comparer à la valeur md5sum qu'il devrait être.

Si vous avez téléchargé le md5sum ainsi que l'ISO, 
Vous pouvez demander au programme md5sum de faire la vérification pour vous. 
Dans le même répertoire:

`md5sum-c ubuntu-10.04-linuxcnc1-i386.iso.md5`

Si tout va bien, après un court délai le terminal affichera:

`ubuntu-10.04-linuxcnc1-i386.iso: OK`

==== Graver l'image ISO sur un CD

 . Inserez un CD vierge dans votre graveur. Une fenêtre surgissante
   «CD/DVD Creator» ou «Choisissez le type de disque» va s'ouvrir. 
   Fermez la, elle ne sera pas utilisée.
 . Naviguez jusqu'au répertoire contenant l'image ISO.
 . Faites un click droit sur le fichier de l'image ISO et choisissez
   Graver le Disque.
 . Sélectionnez la vitesse de gravure. Pour graver le CD Live d'EMC2 il
   est recommandé de graver à la vitesse la plus lente possible pour
   éviter toute erreur de gravure.
 . Lancez la gravure.
 . Si le choix d'un nom de fichier est demandé pour l'image disque,
   cliquez juste OK.

=== Sous Windows

Avant de graver un CD, il est fortement recommandé de vérifier la
somme de contrôle md5 (hash) du fichier de l'image iso.

Windows ne dispose pas de programme de contrôle du md5. Vous devrez en
installer un pour vérifier la somme de contrôle de l'ISO. Plus
d'informations sont disponibles ici:

https://help.ubuntu.com/community/HowToMD5SUM

Gravez l'ISO sur un CD

 . Vous pouvez télécharger Infra Recorder, un logiciel de gravure
   d'images gratuit et open source: http://infrarecorder.org/
 . Inserez un CD vierge dans le graveur sélectionnez 
   Quitter ou Cancel si un auto-run démarre.
 . Lancez Infra Recorder et choisissez le menu 
   'Actions' puis 'Graver l'image'.

== Tester EMC2

Avec le CD Live d'EMC2 dans le lecteur de CD/DVD redémarrez votre PC
de sorte qu'il démarre sur le CD Live. Quand l'ordinateur a redémarré
vous pouvez essayer EMC2 sans l'installer. Vous ne pouvez pas créer de
configuration personnalisée ni modifier les réglages du système comme
la résolution de l'écran sans installer EMC2.

Pour lancer EMC2 allez dans le menu Applications/CNC et choisissez
EMC2. Puis sélectionnez une configuration en sim (simulation) et
essayez le.

Pour savoir si votre ordinateur est utilisable par le générateur de
trains d'impulsions du logiciel, lancez un test de latence comme
indiqué à la section (<<sec:Test-de-latence-GS>>)

== Installer EMC2

Si vous avez envie d'aller plus loin, cliquez juste sur l'icône
Install se trouvant sur le bureau, répondez à quelques questions (votre
nom, votre fuseau horaire, le mot de passe) et faites une installation
complète en quelques minutes. Notez bien le mot de passe indiqué et le
nom d'utilisateur. Une fois l'installation compléte et que vous êtes
connecté, le gestionnaire de mises à jour vous proposera d'effectuer
une mise à jour vers la dernière version stable d'EMC2

== L'interface utilisateur Axis

L'interface AXIS est une des interfaces parmis lesquelles vous avez à
choisir. Elle peut être configurée pour lui ajouter un panneau de
commandes virtuel personnalisé en fonction des besoins. AXIS est
l'interface utilisateur par défaut et est activement développée.

== Les mises à jour d'EMC2 (((Mises à jour d'EMC2)))

Avec l'installation normale de l'Update Manager va vous avertir des mises à jour
d'EMC2 quand vous allez en ligne et vous permettent de facilement mettre à jour sans
Connaissances Linux nécessaires.
Si vous souhaitez passer à 10,04 à partir 8.04 une
installation propre de la Live-CD est recommandée.
Il est OK pour mettre à niveau CEM lorsqu'on lui a demandé d'.

Attention: Ne pas mettre à jour Ubuntu vers une nouvelle version, mais non LTS
(comme 8.4 à 8.10), car elle permettra d'éviter de courir EMC.
